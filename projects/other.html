ReactBot

Date: Nov 2020 - Feb 2021

Role: Programmer

Type: Personal Project

ReactBot is a prototype for a bot that acts as a way for people in different locations to interact with your real world through a robot interface. This can range from turning on and off a desk-lamp to playing music, currently it only send a basic electrical signal as a proof-of-concept.

Programmed a Discord Bot server interface for clients to dynamically connect to using socket in Python 3, and did some internet of things stuff to automatically update both client and the server processes.
