Saucer

Date: Feb 2020

Role: Solo

Type: Class Project

Saucer is a fast-paced dodging game that makes itself unique by limiting how much you can go in each direction at once using a "gas" mechanic. This mechanic would keep things interesting by making it so you could not abuse the spawning velocities of the "asteroids".

Programmed the whole project for my intro to programming midterm, using minimal trigonometry for spawning, and creating unique UI for the title screen that allows the player to be on top of the button to select something.

Etiam

Date: Nov 2017 - Mar 2019

Role: Artist

Type: Personal Project

Etiam is a prototype space shoot-em-up game that was to have a focus on telling a story while also providing fast-paced arcade styled gameplay, bringing modern design practices to the genre.

Created all art assets, ranging from small enemy sprites, large boss sprites, UX, and animations, with dozens of unused sprites and animations that were set to replace many currently used sprites but did not due to our cancellation of the project.
